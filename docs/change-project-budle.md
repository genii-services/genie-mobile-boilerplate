# 패키지와 번들 이름 변경

## react-native-rename

```bash
react-native-rename MyPackageName -b com.company.project
```

## vscode에서 추가 변경 작업

다음 항목을 전체 검색 후 변경

### services.genii.geniemobileboilerplate

어플리케이션 ID로 사용

### genie-mobile-boilerplate

패키지 이름 또는 어플리케이션 이름으로 주로 사용

### Genie Mobile Boilerplate

표시 이름으로 사용

### services/genii/geniemobileboilerplate

안드로이드 네이티브 소스 경로 변경

### GenieMobileBoilerplate

IOS 네이티브 소스 경로 변경
